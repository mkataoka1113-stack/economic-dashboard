Daily learning
PER（株価収益率）
PERは「Price-to-Earnings Ratio」の略で、株価が1株当たりの純利益の何倍になっているかを示す指標です。会社の利益と比べて株価が割高か割安かを判断するのに使われます。PERが低いほど割安と見なされやすいですが、企業の成長性や業界特性によって適正水準は異なります。例えば、ある企業の株価が1000円で1株当たり純利益が100円の場合、PERは10倍（1000円 ÷ 100円）となります。
