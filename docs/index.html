Daily learning
決算（けっさん）
決算とは、企業が一定期間（通常は3ヶ月、6ヶ月、1年）の経営成績や財政状態をまとめることです。投資家は企業の決算発表を通じて、その企業がどれだけ利益を出したか、どれだけの資産を持っているかなどを知り、投資判断の材料とします。例えば、「好決算」とは、企業の利益が予想を上回ったり、成長が著しかったりすることを示し、その企業の株価が上昇する要因となることがあります。
