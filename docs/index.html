Daily learning
PER（株価収益率）
PER（Price Earnings Ratio）は、株価が1株当たり純利益の何倍になっているかを示す指標で、企業の収益力に対して株価が割安か割高かを判断する際に用いられます。計算式は「株価 ÷ 1株当たり純利益」です。例えば、株価が1000円で1株当たり純利益が50円の場合、PERは20倍となります。一般的に、PERが低いほど株価は割安とされ、PERが高いほど株価は割高と判断されることが多いですが、成長性が高い企業は将来の利益成長を織り込み、PERが高くなる傾向もあります。
