Daily learning
スピード調整（株価のスピード調整）
株価が急激に上昇した後、一時的に上昇が一服したり、小幅な下落に転じたりする現象を「スピード調整」と呼びます。これは、株価が短期的に過熱感を示した際に、一部の投資家が利益を確定させる売りを出すことや、新規に買いを入れる投資家が減少することで起こります。株価が持続的に上昇するために必要な、一時的な「休憩」のようなものと捉えられます。例えば、ある企業の株価が好決算発表後に短期間で20%上昇した後、特に悪材料がないにも関わらず数日間で5%程度下落するといった状況がこれに該当します。
