Daily learning
PER（ピーイーアール）
PERとは「Price Earnings Ratio」の略で、日本語では「株価収益率」と訳されます。企業の稼ぐ力に対して、株価が割安なのか割高なのかを判断するための指標です。計算式は「株価 ÷ 1株当たり利益」で、この数値が低いほど割安と判断されることが一般的です。例えば、株価が1000円で1株当たり利益が100円の企業のPERは10倍となります。同業他社のPERと比較することで、その企業の株価水準が適正かどうかを考える手掛かりになります。
