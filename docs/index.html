Daily learning
PER (株価収益率)
PER（Price-to-Earnings Ratio）は、株価が1株当たり純利益の何倍まで買われているかを示す指標で、企業の収益力に対して株価が割安か割高かを判断するのに使われます。PERが低いほど割安と見なされがちですが、成長期待が高い企業はPERが高くなる傾向があります。例えば、1株利益が100円の企業の株価が1,700円の場合、PERは17倍となり、ニュースにあるように市場が割高感を意識する一つの目安となります。
