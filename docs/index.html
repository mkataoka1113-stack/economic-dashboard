Daily Learning

AIバブル
AIバブルとは、人工知能（AI）関連技術や企業への期待が過度に高まり、その株価が企業の実態価値以上に膨れ上がっている状態を指す言葉です。過去のITバブルのように、期待先行で価格が上昇し、その後実態との乖離が顕在化すると急落するリスクが指摘されます。例えば、まだ収益化が難しい段階にあるAI関連スタートアップの株価が、将来の成長期待だけで非常に高値で取引されているような状況がこれに当たります。
