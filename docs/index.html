Daily learning
PER（株価収益率）
PERとは「Price Earnings Ratio」の略で、企業の株価が1株当たりの利益の何倍になっているかを示す指標です。この数字が大きいほど、投資家はその企業の将来性や成長を高く評価していると解釈されますが、同時に株価が割高になっている可能性も示唆します。例えば、1株当たり年間利益が100円の企業の株価が2000円であれば、PERは20倍（2000円 ÷ 100円）となります。市場や同業他社と比較することで、その銘柄が割安か割高かを判断する際の参考になります。
