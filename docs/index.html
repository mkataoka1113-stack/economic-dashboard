Daily learning
暫定関税
暫定関税とは、輸入された特定の品目に対して、一時的に通常の関税よりも高い税率を課す措置のことです。これは、特定の国からの輸入が国内産業に不公正な競争をもたらし、損害を与えている可能性があると判断された場合に、本格的な調査を行う間、国内産業を保護するために導入されます。例えば、中国製の安価な鉄鋼製品が日本市場に大量に流入し、国内の鉄鋼メーカーが苦境に立たされた際、政府が国内産業を守るために暫定的に高い関税を課すことで、輸入価格を上げて競争条件を是正しようとします。
