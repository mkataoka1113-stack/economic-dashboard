Daily learning
PER（株価収益率）
PER（Price Earnings Ratio）は、株価が1株当たりの純利益（EPS）の何倍になっているかを示す指標です。株価の割安・割高を判断する際によく使われます。PERが高いほど、投資家がその企業の将来の成長に対して期待していると解釈されることが多いです。例えば、株価が1000円で1株益が100円の企業であれば、PERは10倍（1000円 ÷ 100円）となります。
