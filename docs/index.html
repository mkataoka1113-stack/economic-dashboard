Today's Term
企業決算
企業決算とは、企業が一定期間（通常3ヶ月または1年）の事業活動の結果をまとめた報告書のことです。この発表は、株価に大きな影響を与えます。例えば、今日のニュースで「ASML決算に注目」とあるように、大手半導体製造装置メーカーのASMLの決算は、関連する半導体セクター全体の株価に影響を与える可能性があります。決算が良いと株価は上がりやすく、悪いと下がりやすい傾向があるため、投資家は決算内容を注意深く見て、投資判断の材料とします。
