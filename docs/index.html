Daily learning
株価収益率（PER）
株価収益率（PER：Price-to-Earnings Ratio）は、株価が1株当たり純利益の何倍になっているかを示す指標です。これは、企業の利益に対して株価が『割安』か『割高』かを判断する目安として使われます。例えば、PERが低い銘柄は割安と見なされやすく、逆にPERが高い銘柄は成長期待が高いものの、割高と判断されることもあります。投資家は、同業他社や過去のPERと比較して、投資判断の材料とすることが一般的です。
