Daily learning
不当廉売関税（アンチダンピング関税）
不当廉売関税とは、外国企業が自国の製品を、国内市場での価格や製造コストよりも不当に低い価格で他国に輸出し、その結果、輸入国の国内産業に損害を与えている場合に課される特別な関税のことです。例えば、海外のA社が日本市場に製品を非常に安い価格で輸出し、日本のB社が価格競争で不利になるような状況を防ぐために、日本政府がこの関税を課すことがあります。これにより、国内産業が不公平な競争から保護されます。
