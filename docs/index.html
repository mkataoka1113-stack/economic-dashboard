Daily learning
自己資本比率規制
銀行が経営の健全性を保つために、どれだけ自分のお金（自己資本）を持っているかを示す比率に関するルールです。この比率が高いほど、銀行は経営が安定しており、万が一の損失が出ても耐えられると判断されます。金融庁がこの規制を緩和するというニュースは、銀行がより積極的に企業へお金を貸したり投資したりしやすくなることを意味し、経済活動が活発になる後押しになる可能性があります。例えば、ベンチャー企業への融資がしやすくなることで、新しい事業が生まれやすくなるといった具体例が挙げられます。
